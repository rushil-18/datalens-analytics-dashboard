# DataLens – Analytics Dashboard

DataLens is a full-stack data analytics dashboard that allows users to upload CSV datasets, store them in PostgreSQL, explore data, generate charts, and compute statistical summaries.

## Features

- Upload CSV datasets
- Store datasets dynamically in PostgreSQL
- Explore dataset tables
- Generate charts
- Compute column statistics
- Delete datasets

## Tech Stack

Backend:
- Python
